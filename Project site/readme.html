Dies ist meine zweite GFS. In dieser Habe ich in Python ein Programm geschrieben, das alle möglichen Werte aller Wiederstände und Teilwiederstände einer Wiederstandschaltung variabler komplexität berechnet und ausgibt.

Eingelesen wird die Schaltung mit einem eigenen simplen Dateiformat.
zur ausgabe wird ein high quality Schaltplan mit Beschriftungen angegeben. Dieser liegt als svg vor. Die Berechneten Werte werden in der GUI danneben ausgegeben.
